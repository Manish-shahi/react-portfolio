# React Portfolio

This is my personal portfolio website built using **React** and **GSAP** for animations. It showcases my projects, skills, and contact information in a modern, interactive interface.

[Add live site link after deployed]

## Features

- Smooth animations with **GSAP**
- Mobile responsive design
- Light & dark mode support
- Interactive components and carousels
- Modern UI using **TailwindCSS** and **Radix UI**

## Tech Stack

- **Frontend:** React 18, Vite, TypeScript
- **Styling:** TailwindCSS v3, tailwindcss-animate
- **Animation:** GSAP
- **Forms & Validation:** react-hook-form, zod
- **Routing:** react-router-dom
- **UI Components:** Radix UI, cmdk, lucide-react
- **Charts & Carousels:** recharts, embla-carousel-react
- **State & Data:** @tanstack/react-query
- **Utilities:** clsx, class-variance-authority, date-fns, lovable-tagger
